Про меня

Меня зовут Марк

С 2014 года занимаюсь преподавательской деятельностью и программированием. Свой путь разработчика я начал с программирования игр на Android Studio, далее перешел на фреймворк Spring backend, моделировал бизнес-процессы с использованием Camunda, в ходе работы над приложениями освоил React/TS . Для меня программирование - это полезный навык и захватывающее занятие.

Frontend
• Разработка enterprise приложений с нуля для кредитного конвейера;
• Настройка локальной среды разработки, сборщика и конфигураций проекта;
• Разработка архитектуры проекта под бизнес требования;
• Сборка и подключение библиотек: UI Kit и для компонентов бизнес логики.
Backend
• Опыт написания контроллеров, сервисов и репозиториев под модель данных, настройка мапперов для обработки JSON, XML и XLS файлов;
• Работа с Camunda: создание делегатов, улучшение bpmn схемы и доработка бизнес-логики в сервисах;
• Опыт интеграционного и модульного тестирования.
Android SDK
• Создание игр на платформе Android в качестве pet-проектов;
• Написание собственного игрового движка;
• Использование паттернов: стратегия, фабричный метод, фасад, адаптер и наблюдатель;
• Работа с потоками, настройкой физической логики и рисованием интерфейса.
Паттерны и алгоритмы
• Работа с алгоритмами и структурами данных: стек, очередь, связанные списки, деревья;
• Оптимизация сортировок, таких как: сортировка вставками, сортировка выбором, сортировка слиянием, пирамидальная сортировка, алгоритм быстрой сортировки.
